Edición gráfica

3.1 ¿Qué es un editor gráfico?

Un editor gráfico es un programa informático que se emplea para crear, modificar, guardar, convertir e imprimir imágenes. Existen diversos tipos de aplicaciones destinadas al tratamiento gráfico, que se pueden agrupar en:

Editores de gráficos vectoriales: Están orientados a la creación y edición de diseños basados en dibujos vectoriales, como logotipos, carteles, etc. Destacan CorIDRAW, LibreOffice Draw.

Editores de imágenes: Tienen como finalidad el tratamiento de imágenes de mapa de bits, tanto dibujos como fotografías.

Software de CAD (computer aided design): Comprende distintas aplicaciones, enfocadas a su uso en ingeniería, arquitectura y diseño. Estas abarcan el modelado, renderizado, la iluminación y creación de gráﬁcos tridimensionales.

Programas de animación. Se usan para crear imágenes 2D o 3D en movimiento, empleando secuencias de fotogramas que se muestran rápidamente. Dar, convertir e imprimir imágenes.

3.3 Formato de imágenes

Formatos para mapa de bits

BMP

Desarrollado para aplicaciones Windows. Al no utilizar compresión, los archivos son muy grandes y no es adecuado para la Web.

GIF

Utiliza 8 bits o 256 colores, y se usa para la creación de animaciones, y se usa para la creación de animaciones, clip art, logotipos o pequeñas imágenes.

JPEG

Muy utilizado en Internet, cámaras de fotos, programas, etc. Usa un algoritmo de compresión con pérdida de calidad.

PNG

Mantiene una alta calidad gráfica en poco espacio, al emplear un algoritmo de compresión sin pérdida. Es uno de los más utilizados.

RAW

Contiene la totalidad de los datos de una imagen, tal como ha sido captada por el sensor digital de la cámara de fotos.

TIFF

Destinado a imágenes en las que su calidad prima con respecto al tamaño de los archivos. Se usa en editoriales y medios impresos.

AI

Desarrollado por Adobe Illustrator para representar logotipos, infografías, gráﬁcos digitales y diseños para impresión.

EPS

Orientado al ámbito editorial, su uso está muy extendido en la Web. Almacena ilustraciones en lenguaje PostScript de alta resolución.

SVG

Optimizado para el diseño web, por lo que se utiliza para logotipos, botones y otros elementos de las páginas web.
